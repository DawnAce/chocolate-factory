Code Splitter 模块是一个代码分割模块，用于将代码分割成多个片段，以便进行各种代码相关任务，如代码相似度计算、代码分类、代码聚类等。

多种不同方式：

- 经典语法分析
  - Antlr
- 基于规则 DSL的语法分析
  - TreeSitter: [https://tree-sitter.github.io/tree-sitter/]
